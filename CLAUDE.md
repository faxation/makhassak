# Claude Code Instructions

## First Step — Every Session

At the start of every new chat or editor session, **always read these files before doing anything else:**

1. `README.md` — Project overview, structure, and tech stack
2. `WEBSITE-CONTEXT.md` — Directives, design decisions, and change log

## After Making Changes

- Add a new entry to the **Change Log** table in `WEBSITE-CONTEXT.md` with the date and a summary of what was changed and why.
